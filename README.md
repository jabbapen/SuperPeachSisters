# SuperPeachSisters

Super marios clone with peach as the main character. Written in C++ using object oriented programming. Use asdw to move left right and jump and space to use a fireball when you get the flower power up. Enemies include Goombas, turtles, and piranha plants. To reach the end of the level get the flag at the end.

To run in Mac run the executable file attached. If you want to make your own download the project, open in XCode and change the assets directory path in main.cpp.
